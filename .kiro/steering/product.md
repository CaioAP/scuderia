# Product Overview

Cilia Scuderia is a modern employee engagement and communication platform built as a web application. The platform focuses on internal company communication, featuring:

## Core Features

- **Employee Dashboard**: Centralized home view with user information and insights
- **Notifications System**: Real-time notifications with read/unread status management
- **Rich Text Messaging**: Quill-based editor for creating and managing messages
- **Employee Insights**: Birthday tracking, announcements, feedback collection, and surveys
- **User Management**: Avatar system and user profiles

## Target Users

Internal employees and HR teams who need a centralized platform for company communication, announcements, and engagement tracking.

## Key Value Propositions

- Streamlined internal communication
- Employee engagement tracking
- Centralized announcement and feedback system
- Modern, responsive user interface
- Real-time notification management
